## Accent color
The main color of the theme is a pastel blue color. It's not much used, but it's the accent color for things like the scrollbar, badges background etc.

- Accent color:
	- Hex Color code: #2D7F9D
	- RGB Color code: R: 45 G: 127 B: 157

## Language colors
The language colors consists of a phew categories. These includes variables, functions, comments etc. JSON is a different categorie as well.

### Comments
The comments are pieces of information that will not be executed.
Styling for the comments is the following:
- Settings
	- Fontstyle: italic
	- Foreground color:
		- Hex Color code: #515555
		- RGB Color code: R: 81 G: 85 B: 85

### Variables
Variables are pieces of code that contain another piece of code. Often times it needs a name, a type and a value.
Styling for the variables is as follows:
- Settings:
	- Foreground color:
		- Hex Color code: #cac68f
		- RGB Color code: R: 202 G: 198 B: 143

### Colors
Colors are other constants.
Styling for the colors:
- Settings:
	- Foreground color:
		- Hex Color code: #c4c7c7
		- RGB Color code: R: 196 G: 199 B: 199

### Invalid
Invalid means a piece of code, that isn't valid or results into a type error. This also includes methods that don't exists.
Styling for invalid is as follows:
- Settings
	- Foreground color:
		- Hex Color code: #b34444
		- RGB Color code: R: 179 G: 68 B: 68
